## 二、记录日历事件（create_calendar_event）

| # | 测试操作 | 期望结果 |

| 2.1 | "小维今天吃了狗粮" | 调 tool，记录 diet 事件，日期=今天，弹出记录卡片 |

| 2.8 | 发送后切到日历页 | 新事件出现在对应日期下 |
