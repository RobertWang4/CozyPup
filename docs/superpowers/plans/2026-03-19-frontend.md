# CozyPup Frontend Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Complete all frontend UX so the app is fully interactive with local data. Every screen, interaction, and flow should work without a backend.

**Architecture:** React 19 + Capacitor iOS. All data stored in localStorage until backend integration. Stores use `useSyncExternalStore` for reactive updates.

**Tech Stack:** React 19, TypeScript, Vite, Capacitor 8, CSS Modules, Lucide icons

**Spec:** `docs/superpowers/specs/2026-03-17-petcare-agent-design.md`
**Design spec:** `docs/superpowers/specs/2026-03-17-frontend-design-prompt.md`
**Design decisions:** See memory `project_spec_decisions.md`

### Task 2: Pet Store

**Files:**
- Create: `frontend/src/stores/petStore.ts`

- [ ] **Step 1: Write pet store with localStorage persistence**

`frontend/src/stores/petStore.ts`:
- `STORAGE_KEY = 'cozypup_pets'`
- `load()` / `save()` — read/write localStorage
- `subscribe()` / `getSnapshot()` — for `useSyncExternalStore`
- `add(data)` — creates Pet with `crypto.randomUUID()`, auto-assigns color from `PET_COLORS[count % 5]`
- `update(id, data)` — merges partial
- `remove(id)` — deletes pet
- `getById(id)` — lookup

Export `usePets()` hook:
```typescript
import { useSyncExternalStore } from 'react';
export function usePets(): Pet[] {
  return useSyncExternalStore(petStore.subscribe, petStore.getSnapshot);
}
```

- [ ] **Step 2: Create haptics utility**

```typescript
import { Capacitor } from '@capacitor/core';

export async function hapticLight() {
  if (!Capacitor.isNativePlatform()) return;
  const { Haptics, ImpactStyle } = await import('@capacitor/haptics');
  await Haptics.impact({ style: ImpactStyle.Light });
}

export async function hapticMedium() {
  if (!Capacitor.isNativePlatform()) return;
  const { Haptics, ImpactStyle } = await import('@capacitor/haptics');
  await Haptics.impact({ style: ImpactStyle.Medium });
}
```
